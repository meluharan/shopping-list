Simple Shopping List With Flat-database Using AJAX/PHP 

The simple shopping list web application was built using AJAX/PHP With flat-database. The application supports multiple shopping lists with functions to add and remove products/items for selected list, as well as adding new or removing an old list. The application stores the state of the list and their contents in a flat-database. The application allows multiple users to add/remove item/list without restriction. 
The application uses also AJAX script to make the system more interactive.
